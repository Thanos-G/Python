# Password Generator

This is a simple password generator program written in Python. It generates a random password of the specified length using a combination of letters (both lowercase and uppercase), digits, and punctuation marks.

## Requirements

- Python 3.x

## Usage

1. Clone the repository or download the `password_generator.py` file.

2. Open a command-line interface and navigate to the directory where the `password_generator.py` file is located.

3. Run the program by executing the following command: python password_generator.py

4. Enter the desired length for the password when prompted.

5. The program will generate a random password of the specified length and display it on the console.

## Example
$ python password_generator.py
Enter the desired length for the password: 12
Generated password: zX5t7f$!Wm2%
